# Чек-лист тестирования AniLibDown 1.1.0

Отмечайте пункты перед релизом. Тестируйте на реальном устройстве, если возможно.

## Установка и запуск

- [ ] Приложение запускается с чистой установки
- [ ] Заставка показывается/скрывается согласно настройке
- [ ] Тема (системная / светлая / тёмная) применяется

## Каталог

- [ ] Список релизов загружается, pull-to-refresh работает
- [ ] Поиск с debounce находит тайтлы
- [ ] История поиска сохраняется и очищается
- [ ] Фильтр по жанрам работает
- [ ] Сортировка и фильтр по году применяются
- [ ] «Случайное» открывает карточку релиза
- [ ] Секция «Продолжить просмотр» показывает незавершённые тайтлы
- [ ] Офлайн: при наличии кеша каталог отображается

## Карточка релиза

- [ ] Метаданные, описание, связанные тайтлы загружаются
- [ ] «Смотреть» продолжает с последней серии
- [ ] Коллекции AniLiberty (добавить / убрать / сменить статус)
- [ ] Блок Shikimori: привязка, смена статуса, отвязка
- [ ] Скачивание одной серии и «Скачать все»

## Плеер

- [ ] Воспроизведение HLS онлайн
- [ ] Смена качества во время просмотра
- [ ] Жесты: пауза, перемотка, ускорение
- [ ] Пропуск OP/ED
- [ ] Автопереход к следующей серии
- [ ] Продолжение с места остановки
- [ ] PiP (свернуть приложение — видео в окне)
- [ ] AirPlay на внешний экран / Apple TV
- [ ] Субтитры (если дорожка есть в потоке)
- [ ] Синхронизация номера серии с Shikimori при досмотре

## Загрузки

- [ ] Загрузка серии, прогресс, отмена
- [ ] Повтор при ошибке
- [ ] Офлайн-воспроизведение скачанного
- [ ] «Только Wi‑Fi» — загрузка не стартует на сотовой сети
- [ ] Лимит параллельных загрузок соблюдается
- [ ] Уведомление о завершении загрузки

## Коллекция и профиль

- [ ] Вход / выход AniLiberty
- [ ] Вкладки коллекции, пагинация
- [ ] Подключение / отключение Shikimori
- [ ] Экспорт и импорт привязок Shikimori
- [ ] Очистка кеша по типам

## iPad

- [ ] Боковая панель с вкладками
- [ ] Каталог и детали в split-view

## Локализация

- [ ] Русский интерфейс по умолчанию
- [ ] Английский при смене языка системы (ключевые экраны)

## Виджет

- [ ] Виджет «Продолжить просмотр» показывает последний тайтл

## Регрессия

- [ ] Unit-тесты проходят (`xcodebuild test`)
- [ ] CI-сборка IPA успешна
- [ ] Нет крашей при быстром переключении вкладок
- [ ] Нет утечки памяти при длительном просмотре (smoke)

## Известные ограничения

- Sideload: срок ~7 дней на бесплатном Apple ID
- Shikimori-привязки локальны (кроме ручного экспорта)
- Виджет требует App Group при сборке с расширением
