# Чек-лист тестирования AniLibDown 1.1.0

Отмечайте пункты перед релизом. Тестируйте на реальном устройстве через Sideloadly.

## Установка и запуск

- [ ] Приложение запускается с чистой установки
- [ ] Заставка показывается/скрывается согласно настройке
- [ ] Тема (системная / светлая / тёмная) применяется

## Каталог

- [ ] Список релизов загружается, pull-to-refresh работает
- [ ] При ошибке сети пустой экран с «Повторить»
- [ ] Поиск с debounce находит тайтлы
- [ ] История поиска сохраняется и очищается свайпом
- [ ] Фильтр по жанрам работает
- [ ] Сортировка и фильтр по году (валидный год, например 2024)
- [ ] «Случайное» открывает карточку релиза
- [ ] «Продолжить просмотр» — постер и название после просмотра серии

## Карточка релиза

- [ ] Метаданные, описание, связанные тайтлы
- [ ] «Повторить» при ошибке загрузки карточки
- [ ] «Смотреть» продолжает с последней серии
- [ ] Коллекции AniLiberty
- [ ] Блок Shikimori: привязка, статус, отвязка
- [ ] Скачивание серии / «Скачать все»

## Плеер

- [ ] Воспроизведение онлайн и офлайн
- [ ] Toast, если видео недоступно
- [ ] Смена качества, жесты, OP/ED, автоследующая
- [ ] PiP / AirPlay (если устройство поддерживает)

## Загрузки

- [ ] Прогресс, отмена, повтор
- [ ] «Только Wi‑Fi» — toast, без фантомных элементов
- [ ] «Очистить кеш» неактивна, когда список пуст
- [ ] Уведомление о завершении (после первого запроса разрешения)

## Коллекция и профиль

- [ ] Вход / выход AniLiberty
- [ ] Ошибка коллекции с «Повторить» / баннер
- [ ] Экспорт/импорт Shikimori с иконками
- [ ] Версия вида `1.1.0 (1)` в профиле

## Регрессия

- [ ] Actions → Build IPA зелёный
- [ ] Нет крашей при быстром переключении вкладок

## Известные ограничения Sideloadly

- Срок ~7 дней на бесплатном Apple ID
- Виджеты Home Screen недоступны без Xcode + платной подписи
- PiP зависит от устройства; при сбое будет toast
