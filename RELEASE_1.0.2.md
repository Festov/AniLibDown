# AniLibDown 1.0.2 — что сделано

## Исправления

- **Коллекции:** pull-to-refresh больше не очищает список до ответа API и не «зависает»; при ошибке сети старые элементы сохраняются
- **Сборка:** исправлены ошибки типов в миграции настройки ускорения и `await` для `CHHapticEngine.start()`

## Карточка тайтла

- Тап по постеру → полноэкранный просмотр с pinch-zoom
- Кнопки **«Смотреть»** и **«Коллекции»** в одной строке; в кнопке коллекций сразу виден статус (Смотрю / План / …)
- Кнопки выше и красного цвета AniLibria (акцент приложения)
- Блок **«Связанные»** перенесён под описание; названия выровнены по левому краю
- Выбор качества убран с карточки — используется значение из профиля

## Профиль

- Внизу профиля отображается номер версии **1.0.2**
- **Качество по умолчанию** (1080p / 720p / 480p) — для просмотра и скачивания
- **Ускорение при удержании** — **1,5×** или **2×**
- Переключатель **«Показывать Shikimori»** перенесён в секцию Shikimori
- Пояснение: привязки Shikimori хранятся локально на телефоне и слетят при переустановке

## Плеер

- Зажатие правой половины экрана ускоряет видеоряд (скорость из настроек)
- Бейдж ускорения отображается **по центру, под названием серии** (не под пальцем)
- Настройки плеера (шестерёнка) — **одна секция** без деления на «Перемотка» / «Воспроизведение»

## Заставка

- При появлении надписи «Спасибо, что выбираете нас!» — **одна непрерывная вибрация 0,6 с** (тип как у звонка/будильника, через Core Haptics)
- Усиленные фоновые эффекты: переливание, орбы, волна, частицы, второе вращающееся кольцо

## Загрузки

- Во время скачивания кнопка скачивания заменяется **кнопкой отмены** с прогрессом по кругу
- После завершения убрана зелёная галочка — остаётся только иконка удаления
- Корректное склонение «серия / серии / серий» в списке загрузок
- Постеры скачанных тайтлов сохраняются **локально** и отображаются без интернета
