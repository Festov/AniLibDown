# Чек-лист тестирования AniLibDown 1.0.2

Проверять на реальном iPhone (хаптика и офлайн-постеры на симуляторе ограничены).

## Коллекции

- [ ] Открыть вкладку «Коллекция», выбрать тип с элементами
- [ ] Потянуть список вниз (pull-to-refresh): список **не** должен обнуляться и зависать
- [ ] После refresh элементы остаются / обновляются, повторный swipe снова работает
- [ ] Пустая коллекция: после refresh можно снова потянуть для обновления
- [ ] При ошибке сети во время refresh старые элементы не пропадают
- [ ] Переключение типа коллекции по-прежнему работает

## Карточка тайтла

- [ ] Тап по постеру открывает полноэкранный просмотр
- [ ] В полноэкранном режиме работает pinch-zoom и закрытие (тап / крестик)
- [ ] Кнопки «Смотреть» и «Коллекции» в одной строке, красного цвета AniLibria, удобной высоты
- [ ] В кнопке «Коллекции» сразу виден статус (Смотрю / План / … или «Коллекции»)
- [ ] Смена статуса через меню коллекции работает
- [ ] Без авторизации тап по «Коллекции» открывает вход
- [ ] Блок «Связанные» находится **под** описанием
- [ ] Названия в «Связанных» выровнены **по левому краю**
- [ ] Выбор качества **убран** с карточки (берётся из профиля)

## Профиль / настройки

- [ ] Внизу профиля отображается номер версии `1.0.2`
- [ ] Есть выбор «Качество по умолчанию» (1080p / 720p / 480p)
- [ ] Выбранное качество используется при «Смотреть» и скачивании
- [ ] Есть выбор «Ускорение при удержании»: **1,5×** и **2×**
- [ ] Переключатель «Показывать Shikimori» находится в секции **Shikimori** (с пояснением про локальное хранение)

## Плеер

- [ ] Зажатие правой половины экрана ускоряет видеоряд
- [ ] Скорость соответствует настройке из профиля / настроек плеера (1,5× или 2×)
- [ ] Бейдж ускорения отображается **по центру, под названием серии** (не под пальцем справа)
- [ ] После отпускания скорость возвращается к обычной
- [ ] В настройках плеера (шестерёнка) все пункты в **одной секции**

## Заставка

- [ ] При появлении надписи «Спасибо, что выбираете нас!» срабатывает **одна** вибрация ~**0,6 с** (на устройстве)
- [ ] Фон заставки: переливание, орбы, волна, частицы
- [ ] Если заставка выключена в профиле — не показывается

## Загрузки

- [ ] При скачивании серии кнопка скачивания **заменяется** кнопкой отмены
- [ ] Прогресс отображается **кольцом вокруг** кнопки отмены
- [ ] Отмена останавливает загрузку и возвращает кнопку скачивания
- [ ] После завершения **нет** зелёной галочки — только иконка удаления
- [ ] В списке загрузок склонение корректно: 1 серия, 2–4 серии, 5+ серий (и 11–14 серий)
- [ ] Скачать несколько серий тайтла → включить авиарежим → вкладка «Загрузки»: постер отображается **без сети**
- [ ] Новая загрузка сохраняет постер локально (после первого успешного скачивания)

## Регрессия

- [ ] Новинки / каталог / поиск открываются
- [ ] Онлайн-просмотр серии работает
- [ ] Офлайн-просмотр скачанной серии работает
- [ ] Shikimori: привязка и смена статуса (если настроено)
- [ ] Тема светлая/тёмная/системная
- [ ] Удаление загрузки / очистка кеша не ломают приложение
